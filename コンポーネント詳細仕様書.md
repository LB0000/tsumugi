# コンポーネント詳細仕様書

## 1. スタイル選択モーダル

ユーザーがアートスタイルを選択するためのモーダルコンポーネントです。

### 構造

```
StyleModal
├── ModalOverlay (半透明の黒背景)
├── ModalContent
│   ├── ModalHeader ("Choose Your Style" + 閉じるボタン)
│   ├── StyleList
│   │   ├── StyleCard (Intelligent) - デフォルト選択
│   │   ├── StyleCard (Baroque Red)
│   │   ├── StyleCard (Florentine Renaissance)
│   │   ├── StyleCard (Renaissance Sky)
│   │   ├── StyleCard (Rococo)
│   │   └── ... (その他のスタイル)
│   └── ConfirmButton
```

### StyleCardコンポーネント

各スタイルカードには以下の要素を含めます。

| 要素 | 説明 |
|------|------|
| サムネイル画像 | そのスタイルで生成されたサンプル画像（80x80px程度） |
| スタイル名 | 太字で表示（例：「Baroque Red」） |
| 説明文 | スタイルの特徴を1〜2行で説明 |
| カラーパレット | 4〜5個の小さな円形カラーチップで色調を表示 |
| 選択状態 | 選択時は緑色のチェックマークとボーダーを表示 |

### スタイルデータ構造

```typescript
interface ArtStyle {
  id: string;
  name: string;
  description: string;
  thumbnailUrl: string;
  colorPalette: string[]; // HEXカラーコードの配列
  promptKeywords: string; // AI生成時に使用するキーワード
}
```

## 2. 画像アップロードコンポーネント

### 構造

```
ImageUploader
├── DropZone (点線ボーダーのエリア)
│   ├── UploadIcon
│   ├── MainText ("Upload Photo")
│   ├── SubText (ヒントテキスト、動的に変化)
│   └── HiddenFileInput
├── PickStyleButton (右側に配置)
└── ProgressIndicator (アップロード中のみ表示)
```

### 動的ヒントテキスト

カテゴリに応じてヒントテキストを変更します。

| カテゴリ | ヒントテキスト |
|----------|----------------|
| Pets | "Show your pet's full face" または "Use a well-lit photo" |
| Family | "Upload one or more photos – people and pets welcome" |
| Kids | "Upload one or more photos – siblings and pets welcome" |

### 状態管理

```typescript
interface UploadState {
  status: 'idle' | 'uploading' | 'processing' | 'complete' | 'error';
  progress: number; // 0-100
  previewUrl: string | null;
  errorMessage: string | null;
}
```

## 4. プログレスインジケーター

ページ上部に表示される3ステップのインジケーターです。

### 構造

```
ProgressIndicator
├── Step (Upload) - アイコン + テキスト
├── Connector (矢印または線)
├── Step (Preview) - アイコン + テキスト
├── Connector
└── Step (Download or Order Print) - アイコン + テキスト
```

### 状態表示

| 状態 | スタイル |
|------|----------|
| 完了 | 緑色のチェックマーク、テキストは白 |
| 現在 | 白いテキスト、下線付き |
| 未完了 | グレーのテキスト |

## 7. アニメーション仕様

### ホバーエフェクト

| 要素 | エフェクト |
|------|------------|
| ボタン | 背景色が少し明るくなる（opacity変化）、0.2秒のトランジション |
| カード | 軽い浮き上がり効果（translateY: -2px）、影の追加 |
| サンプル画像 | スケール拡大（scale: 1.02）、0.3秒のトランジション |

### モーダル表示

| 段階 | アニメーション |
|------|----------------|
| 表示時 | フェードイン（opacity 0→1）+ スライドアップ（translateY: 20px→0） |
| 非表示時 | フェードアウト + スライドダウン |
| 所要時間 | 0.3秒 |

## 8. レスポンシブブレークポイント

| ブレークポイント | 幅 | 主な変更点 |
|------------------|-----|------------|
| モバイル | 〜640px | サイドバーは非表示（ハンバーガーメニューで展開）、1カラムレイアウト |
| タブレット | 641px〜1024px | サイドバーは非表示、2カラムグリッド |
| デスクトップ | 1025px〜 | サイドバー常時表示可能、3カラムグリッド |
