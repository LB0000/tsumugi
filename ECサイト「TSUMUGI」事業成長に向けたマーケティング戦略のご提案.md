# ECサイト「TSUMUGI」事業成長に向けたマーケティング戦略のご提案

## はじめに

この度は、AI肖像画専門ECサイト「TSUMUGI」のマーケティング戦略立案という重要な機会をいただき、誠にありがとうございます。本提案書は、貴社の現状分析データと外部の市場調査に基づき、売上とブランド価値を最大化するための具体的かつ実行可能な戦略を提示するものです。

私どもは、本提案が貴社の事業を必ず成功へと導く一助となることを確信しております。

まず、TSUMUGIが置かれている市場環境を整理し、事業機会を分析します。

### 1.1. 関連市場の規模と成長性

TSUMUGIが関連する市場は、いずれも堅調な成長が見込まれる有望な領域です。

| **ギフト市場（日本）** | 10兆円規模 [2] | 安定した巨大市場。近年は、体験や個人の好みに合わせた「パーソナライズギフト」の需要が高まっています。 |

### 1.2. 競合環境とTSUMUGIの立ち位置

次に、競合環境を分析し、TSUMUGIの独自の強みと克服すべき課題を明確にします。

**主要競合「Pet Canva」との比較**

| 項目 | 紡 TSUMUGI | Pet Canva |
| :--- | :--- | :--- |
| **コンセプト** | AIと日本の美意識の融合 | 手描きアーティストによるポップアート |
| **価格** | 2,900円〜（デジタル） | 8,980円〜（物理商品セット） |

**分析から導かれるTSUMUGIの機会と課題**

*   **機会**: Pet Canvaが「手描き・高価格帯」であるのに対し、TSUMUGIは**「AIによる手軽さ・低価格」**という明確な差別化が可能です。特に、パーソナライズギフト市場の成長を取り込む上で、この手軽さは大きな武器となります。
*   **課題**: 「AIが自動で生成する」という点は、温かみや特別感を求める顧客層から敬遠される可能性があります。また、現状はデジタルデータの販売が中心であり、物理的な「贈り物」としての価値を提供しきれていない点が機会損失につながっています。

以上の分析を踏まえ、次章以降で具体的な戦略を提案します。

## 4. サイト内改善（コンバージョン率最適化）

集客した顧客を確実に購入へと導くため、サイトの課題を改善し、購入体験を向上させる必要があります。日本のECサイトの平均コンバージョン率（購入率）が1〜2%であるのに対し[4]、ギフトやヘルスケアといった専門性の高い分野では4%を超える事例もあり[5]、TSUMUGIには大きな改善の余地があります。

### 4.1. トップページの改善

トップページはサイトの顔であり、顧客が最初に訪れる場所です。ここで興味を惹きつけ、信頼感を与え、次の行動を促すことが極めて重要です。

| 課題点 | 具体的な改善案 |
| :--- | :--- |
| **情報が多すぎる・散漫** | ・**ファーストビューの再構成**: 現在はキャンペーン情報、カテゴリ、メインコピー、CTAが混在し、視線が分散しています。最も伝えたいメッセージである「AIが描く、あなただけの肖像画」と「今すぐ作成する」ボタンを中央に大きく配置し、他の要素は整理・縮小します。
・**3ステップの視覚化**: 「写真アップロード→スタイル選択→生成」の流れを、より直感的で分かりやすいイラストやアイコンを用いて視覚的に表現し、利用の簡単さを強調します。 |
| **ファミリー・キッズの事例不足** | ・**カテゴリ別ギャラリーの設置**: 現在はペットの事例しかありません。「ファミリー」「キッズ」のタブをタップすると、それぞれのカテゴリの魅力的な作例が複数表示されるようにUIを改修します。これにより、ペットオーナー以外の顧客層も自分事として捉えやすくなります。 |
| **レビューの具体性欠如** | ・**顧客の声（UGC）の掲載**: 「4.9 (2,847件)」という数字だけでなく、実際に購入した顧客からの具体的なコメントと、作成された肖像画をセットで掲載します。「想像以上のクオリティでした！」「友人へのプレゼントにとても喜ばれました」といったリアルな声は、何よりの信頼証明となります。 |

### 4.2. 商品選択・決済フローの改善

購入意欲が高まった顧客を、迷わせることなくスムーズに決済完了まで導くための改善案です。

| 課題点 | 具体的な改善案 |
| :--- | :--- |
| **価格体系の不透明さ** | ・**料金プランページの作成**: 「STARTER」「STUDIO」といったプランの違いが現状では不明確です。それぞれのプランで選択できるアートスタイルの種類、解像度、納品形式などを明確に比較した料金プランページを新設します。価格の違いに対する納得感を醸成し、アップセルを促進します。 |
| **ギフト利用時のハードル** | ・**eギフト機能の導入**: 相手の住所を知らなくても、LINEやメールでギフトを贈れる「eギフト機能」を導入します。これにより、ペルソナ3「気の利いたギフトを探す人」のサプライズ需要を確実に取り込み、コンバージョン率を大幅に向上させます。スープストックトーキョーの事例では、eギフト購入者の70%が新規顧客であったというデータもあります[6]。 |
| **カゴ落ち対策** | ・**カゴ落ちリマインドメール**: 商品をカートに入れたまま離脱してしまった顧客に対し、24時間後などに「お忘れ物はありませんか？」といったリマインドメールを自動送信します。カートに入れた商品の画像を表示し、購入を再度促します。 |

## 5. リピート促進・LTV最大化戦略

新規顧客の獲得コストは、既存顧客の維持コストの5倍かかると言われています（1:5の法則）。事業の持続的な成長のためには、一度購入した顧客との関係を維持し、顧客生涯価値（LTV）を高めていく戦略が不可欠です。

### 5.1. 顧客との関係構築施策

購入後も顧客との接点を持ち続け、TSUMUGIブランドへの愛着を育てるための施策です。

| 施策 | 具体的なアクション |
| :--- | :--- |
| **LINE公式アカウントへの誘導と活用** | ・**購入完了ページでの登録促進**: 購入完了ページで「LINEで作成した肖像画を受け取る」「LINE限定クーポンをゲット」といったメリットを提示し、LINE公式アカウントへの友だち登録を促します。
・**ステップ配信**: 友だち登録後、①お礼メッセージ、②肖像画の活用アイデア（SNS投稿、スマホ壁紙設定など）、③次回使えるクーポン、④友人紹介プログラムの案内、といった内容を段階的に配信し、顧客エンゲージメントを高めます。 |
| **記念日リマインドと特別オファー** | ・**記念日登録機能**: 購入時やマイページで、ペットの誕生日や家族になった日などを登録してもらいます。
・**パーソナライズドDM**: 記念日の1ヶ月前などに、「まもなく〇〇ちゃん（ペット名）の誕生日ですね！新しいアートスタイルで記念の肖像画はいかがですか？」といったパーソナルなメッセージと共に、特別クーポンをメールやLINEで配信し、リピート購入を喚起します。 |

### 5.2. アップセル・クロスセル戦略

顧客単価を引き上げ、LTVを最大化するための具体的な商品・サービスの提案です。

| 施策 | 具体的なアクション |
| :--- | :--- |
| **物理グッズの販売（クロスセル）** | ・**肖像画データのグッズ化**: デジタルデータ納品後、作成した肖像画を使って「キャンバスプリント」「マグカップ」「スマホケース」「Tシャツ」などのオリジナルグッズを作成・購入できるサービスを導入します。これにより、デジタル（2,900円）のみの購入から、物理的な商品（＋3,000円〜）へのクロスセルを狙い、顧客単価を大幅に向上させます。これは、競合が展開している成功モデルでもあります。
・**ギフトセットの造成**: 「肖像画データ＋キャンバスプリント＋メッセージカード」のようなギフトセットを用意し、「贈り物」としての価値を高めます。 |
| **高価格帯アートスタイルの提案（アップセル）** | ・**限定・著名アーティストコラボスタイル**: 有名イラストレーターや画家と提携し、「〇〇（アーティスト名）風スタイル」といった高価格帯（例: 9,800円〜）の限定アートスタイルを開発します。希少性と付加価値を高め、特別な記念品を求める顧客層の需要に応えます。 |
| **友人紹介プログラム** | ・**インセンティブの提供**: 既存顧客が友人にTSUMUGIを紹介し、その友人が購入した場合、紹介者と友人の両方に次回使える割引クーポン（例: 1,000円OFF）を付与します。口コミによる新規顧客獲得と、既存顧客のリピート利用を同時に促進する、非常に費用対効果の高い施策です。 |

## 7. KPI設定と効果測定

提案した各施策の成果を客観的に評価し、継続的な改善サイクルを回すために、以下の重要業績評価指標（KPI）を設定します。

### 7.1. 主要KPI

事業全体の健全性を測るための最重要指標です。これらは月次で定点観測し、経営判断の基礎とします。

| KPI項目 | 目標値（6ヶ月後） | 測定方法・ツール |
| :--- | :--- | :--- |
| **月間売上高** | 500万円 | ECシステム売上データ |
| **新規顧客獲得数** | 1,500人/月 | Google Analytics、ECシステム顧客データ |
| **顧客獲得単価（CPA）** | 2,000円以内 | 各広告媒体の管理画面、Google Analytics |
| **サイト全体のコンバージョン率（CVR）** | 2.5% | Google Analytics |
| **顧客生涯価値（LTV）** | 8,000円（12ヶ月） | CRM/顧客管理システムにて算出（平均顧客単価 × 平均購入回数） |

*（注）目標値は、提案施策が計画通りに進んだ場合の推計です。CPAの目標値は、想定LTV（8,000円）と粗利率（仮に80%と設定）から、許容CPAを6,400円とした上で、利益確保のために戦略的に設定しています。*

### 7.2. 施策別KPI

各マーケティング施策のパフォーマンスを個別に評価するための指標です。

| 施策カテゴリ | KPI | 目標値 | 測定方法 |
| :--- | :--- | :--- | :--- |
| **集客戦略** | Instagram広告のクリック単価（CPC） | 80円以下 | Instagram広告マネージャー |
| | コンテンツSEO経由の自然検索流入数 | 5,000セッション/月 | Google Search Console |
| **サイト内改善** | トップページから商品作成ページへの遷移率 | 40% | Google Analytics |
| | 決済フローにおける離脱率 | 15%未満 | Google Analytics |
| **リピート促進** | LINE公式アカウントの友だち追加率（購入者比） | 30% | LINE Official Account Manager |
| | 顧客リピート率（購入後3ヶ月以内） | 20% | ECシステム顧客データ |

### 7.3. 運用と改善サイクル

これらのKPIを絵に描いた餅にしないため、以下の体制でPDCA（計画・実行・評価・改善）サイクルを回します。

1.  **週次定例ミーティング**: 各施策の担当者がKPIの進捗を報告し、短期的な課題と改善策を議論します。
2.  **月次レビュー**: 主要KPIの達成度を経営陣を含めてレビューし、翌月の戦略や予算配分を決定します。
3.  **ダッシュボードの構築**: Google Analyticsや各広告媒体のデータを統合したダッシュボードを構築し、KPIの状況をリアルタイムで可視化します。これにより、迅速な意思決定を可能にします。

このサイクルを通じて、データに基づいた客観的な判断を下し、戦略の精度を継続的に高めていきます。

## 8. 実行計画（ロードマップ）

提案した戦略を確実に実行し、成果を最大化するため、以下のロードマップを提案します。施策を「短期」「中期」「長期」の3つのフェーズに分け、優先順位を明確にしました。

### 短期（1〜3ヶ月）：基盤構築と即時改善

このフェーズでは、低コストで着手でき、すぐに成果が見込めるサイト内改善と、主要な集客チャネルの立ち上げに集中します。

| 優先度 | 施策 | 担当部署（想定） | 主要KPI |
| :--- | :--- | :--- | :--- |
| **高** | **サイト内改善**（トップページ改善、料金プランページ作成、レビュー掲載） | Web制作チーム、マーケティング | CVR、離脱率 |
| **高** | **Instagram/TikTok広告の開始** | マーケティング | CPA、CPC、新規顧客獲得数 |
| **中** | **Google検索広告の開始** | マーケティング | CPA、CVR |
| **中** | **LINE公式アカウントの開設と購入後フローへの導入** | マーケティング | 友だち追加率 |

### 中期（4〜12ヶ月）：顧客関係の深化と収益源の多様化

短期施策で構築した基盤の上で、顧客との長期的な関係を築き、LTVを向上させる施策と、新たな収益源となる物理グッズの展開を進めます。

| 優先度 | 施策 | 担当部署（想定） | 主要KPI |
| :--- | :--- | :--- | :--- |
| **高** | **物理グッズ（キャンバスプリント等）の販売開始** | 商品開発、Web制作チーム | LTV、顧客単価 |
| **高** | **コンテンツSEO（ブログ記事）の本格展開** | マーケティング、外部ライター | 自然検索流入数、新規顧客獲得数 |
| **中** | **eギフト機能の導入** | Web制作チーム、システム開発 | CVR、ギフト利用率 |
| **中** | **友人紹介プログラムの開始** | マーケティング | 紹介経由の新規顧客数 |
| **低** | **ペット系インフルエンサーマーケティングの実施** | マーケティング | エンゲージメント率、CPA |

### 長期（13ヶ月〜）：事業のスケールとブランド確立

事業をさらに拡大し、市場における確固たる地位を築くための、より大きな投資を伴う施策に着手します。

| 優先度 | 施策 | 担当部署（想定） | 主要KPI |
| :--- | :--- | :--- | :--- |
| **高** | **法人向け（BtoB）サービスの開発・営業開始** | 事業開発、営業 | 提携企業数、法人売上 |
| **中** | **ペットメモリアル事業への参入（葬儀社等との提携）** | 事業開発、商品開発 | 提携社数、メモリアル関連売上 |
| **中** | **オフラインイベントへの出展** | マーケティング、営業 | ブース来場者数、リード獲得数 |
| **低** | **AI技術を活用した新機能（動画生成等）の開発** | 開発チーム | 新機能利用率、顧客満足度 |
